CDAPI

The CDApi (Clinical Document Architecture Programming Interface) is an object-oriented tool for Java, created to assist developers of Health Information Systems (HIS) software. The adoption of this tool has the objective of making the processes between the systems responsible for the operationalization of the HL7CDA standard more agile, providing improvements in the access to updated medical records of patients. Through this API, applications can transfer clinical data between doctors and patients. This API is currently available in version 1.0. It consists of a series of functions accessible only by programming and can be downloaded through the link provided with instructions on how to use the tool by customers.
