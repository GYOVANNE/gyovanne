MultCare Médico

Software web

O MultCare Médico é uma solução para a manipulação e compartilhamento de documentos clínicos baseados no padrão HL7 CDA utilizada para o diagnóstico da DRC.

Radio Retrô

Software web

A RádioRetrô é um reprodutor de áudio disponível para usuários que gostam de ouvir músicas de qualidade. No geral, esse sistema fornece ao usuário as funcionalidades básicas de um reprodutor de áudio, com uma interface simplificada e bem intuitiva.

Solar System

Sistema web

O Solar System é um software para web para amantes de astronomia. Nele é possível visualizar o sistema solar de maneira precisa, na perspectiva de velocidade, distância e tamanho de cada planeta.
